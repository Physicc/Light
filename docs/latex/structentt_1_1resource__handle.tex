\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Resource$>$\newline
struct entt\+::resource\+\_\+handle$<$ Resource $>$}

Shared resource handle. 

A shared resource handle is a small class that wraps a resource and keeps it alive even if it\textquotesingle{}s deleted from the cache. It can be either copied or moved. A handle shares a reference to the same resource with all the other handles constructed for the same identifier.~\newline
 As a rule of thumb, resources should never be copied nor moved. Handles are the way to go to keep references to them.

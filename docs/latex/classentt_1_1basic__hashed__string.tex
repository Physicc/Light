\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Char$>$\newline
class entt\+::basic\+\_\+hashed\+\_\+string$<$ Char $>$}

Zero overhead unique identifier. 

A hashed string is a compile-\/time tool that allows users to use human-\/readable identifers in the codebase while using their numeric counterparts at runtime.~\newline
 Because of that, a hashed string can also be used in constant expressions if required.

\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Derived, typename Delta$>$\newline
class entt\+::process$<$ Derived, Delta $>$}

Base class for processes. 

This class stays true to the C\+R\+TP idiom. Derived classes must specify what\textquotesingle{}s the intended type for elapsed times.~\newline
 A process should expose publicly the following member functions whether required\+:


\begin{DoxyItemize}
\item 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordtype}{void} update(Delta, \textcolor{keywordtype}{void} *);}
\end{DoxyCode}


It\textquotesingle{}s invoked once per tick until a process is explicitly aborted or it terminates either with or without errors. Even though it\textquotesingle{}s not mandatory to declare this member function, as a rule of thumb each process should at least define it to work properly. The {\ttfamily void $\ast$} parameter is an opaque pointer to user data (if any) forwarded directly to the process during an update.
\item 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordtype}{void} init();}
\end{DoxyCode}


It\textquotesingle{}s invoked when the process joins the running queue of a scheduler. This happens as soon as it\textquotesingle{}s attached to the scheduler if the process is a top level one, otherwise when it replaces its parent if the process is a continuation.
\item 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordtype}{void} succeeded();}
\end{DoxyCode}


It\textquotesingle{}s invoked in case of success, immediately after an update and during the same tick.
\item 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordtype}{void} failed();}
\end{DoxyCode}


It\textquotesingle{}s invoked in case of errors, immediately after an update and during the same tick.
\item 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keywordtype}{void} aborted();}
\end{DoxyCode}


It\textquotesingle{}s invoked only if a process is explicitly aborted. There is no guarantee that it executes in the same tick, this depends solely on whether the process is aborted immediately or not.
\end{DoxyItemize}

Derived classes can change the internal state of a process by invoking the {\ttfamily succeed} and {\ttfamily fail} protected member functions and even pause or unpause the process itself.

\doxysubsection{Detailed Description}
Function object to wrap {\ttfamily std\+::sort} in a class type. 

Unfortunately, {\ttfamily std\+::sort} cannot be passed as template argument to a class template or a function template.~\newline
 This class fills the gap by wrapping some flavors of {\ttfamily std\+::sort} in a function object.

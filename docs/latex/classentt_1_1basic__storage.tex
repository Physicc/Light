\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Entity, typename Type, typename = void$>$\newline
class entt\+::basic\+\_\+storage$<$ Entity, Type, typename $>$}

Basic storage implementation. 

This class is a refinement of a sparse set that associates an object to an entity. The main purpose of this class is to extend sparse sets to store components in a registry. It guarantees fast access both to the elements and to the entities.

\begin{DoxyNote}{Note}
Entities and objects have the same order. It\textquotesingle{}s guaranteed both in case of raw access (either to entities or objects) and when using random or input access iterators.

Internal data structures arrange elements to maximize performance. There are no guarantees that objects are returned in the insertion order when iterate a storage. Do not make assumption on the order in any case.
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Empty types aren\textquotesingle{}t explicitly instantiated. Therefore, many of the functions normally available for non-\/empty types will not be available for empty ones.
\end{DoxyWarning}

Sort elements according to the given comparison function. 

The comparison function object must return {\ttfamily true} if the first element is {\itshape less} than the second one, {\ttfamily false} otherwise. The signature of the comparison function should be equivalent to one of the following\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{bool(\textcolor{keyword}{const} Entity, \textcolor{keyword}{const} Entity);}
\DoxyCodeLine{bool(\textcolor{keyword}{const} Type \&, \textcolor{keyword}{const} Type \&);}
\end{DoxyCode}


Moreover, the comparison function object shall induce a {\itshape strict weak ordering} on the values.

The sort function oject must offer a member function template {\ttfamily operator()} that accepts three arguments\+:


\begin{DoxyItemize}
\item An iterator to the first element of the range to sort.
\item An iterator past the last element of the range to sort.
\item A comparison function to use to compare the elements.
\end{DoxyItemize}

\begin{DoxyWarning}{Warning}
Empty types are never instantiated. Therefore, only comparison function objects that require to return entities rather than components are accepted.
\end{DoxyWarning}

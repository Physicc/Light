\subsubsection*{template$<$typename Entity$>$\newline
class entt\+::basic\+\_\+snapshot\+\_\+loader$<$ Entity $>$}

Utility class to restore a snapshot as a whole. 

A snapshot loader requires that the destination registry be empty and loads all the data at once while keeping intact the identifiers that the entities originally had.~\newline
 An example of use is the implementation of a save/restore utility.

Before the inclusion of \mbox{\hyperlink{imgui_2examples_2libs_2glfw_2include_2GLFW_2glfw3native_8h}{glfw3native.\+h}}, you may define zero or more window system A\+PI macro and zero or more context creation A\+PI macros.

The chosen backends must match those the library was compiled for. Failure to do this will cause a link-\/time error.

\subsubsection*{template$<$typename Ret, typename... Args$>$\newline
class entt\+::sigh$<$ Ret(\+Args...)$>$}

Unmanaged signal handler. 

It works directly with references to classes and pointers to member functions as well as pointers to free functions. Users of this class are in charge of disconnecting instances before deleting them.

This class serves mainly two purposes\+:


\begin{DoxyItemize}
\item Creating signals to use later to notify a bunch of listeners.
\item Collecting results from a set of functions like in a voting system.
\end{DoxyItemize}

Collects return values from the listeners. 

The collector must expose a call operator with the following properties\+:


\begin{DoxyItemize}
\item The return type is either {\ttfamily void} or such that it\textquotesingle{}s convertible to {\ttfamily bool}. In the second case, a true value will stop the iteration.
\item The list of parameters is empty if {\ttfamily Ret} is {\ttfamily void}, otherwise it contains a single element such that {\ttfamily Ret} is convertible to it.
\end{DoxyItemize}

\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Concept, std\+::size\+\_\+t Len, std\+::size\+\_\+t Align$>$\newline
class entt\+::basic\+\_\+poly$<$ Concept, Len, Align $>$}

Static polymorphism made simple and within everyone\textquotesingle{}s reach. 

Static polymorphism is a very powerful tool in C++, albeit sometimes cumbersome to obtain.~\newline
 This class aims to make it simple and easy to use.

\begin{DoxyNote}{Note}
Both deduced and defined static virtual tables are supported.~\newline
 Moreover, the {\ttfamily poly} class template also works with unmanaged objects.
\end{DoxyNote}

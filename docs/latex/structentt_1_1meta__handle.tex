\doxysubsection{Detailed Description}
Opaque pointers to instances of any type. 

A handle doesn\textquotesingle{}t perform copies and isn\textquotesingle{}t responsible for the contained object. It doesn\textquotesingle{}t prolong the lifetime of the pointed instance.~\newline
 Handles are used to generate references to actual objects when needed.

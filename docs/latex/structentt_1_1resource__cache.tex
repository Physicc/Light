Simple cache for resources of a given type. 

Minimal implementation of a cache for resources of a given type. It doesn\textquotesingle{}t offer much functionalities but it\textquotesingle{}s suitable for small or medium sized applications and can be freely inherited to add targeted functionalities for large sized applications.

Loads the resource that corresponds to a given identifier. 

In case an identifier isn\textquotesingle{}t already present in the cache, it loads its resource and stores it aside for future uses. Arguments are forwarded directly to the loader in order to construct properly the requested resource.

\begin{DoxyNote}{Note}
If the identifier is already present in the cache, this function does nothing and the arguments are simply discarded.
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
If the resource cannot be loaded correctly, the returned handle will be invalid and any use of it will result in undefined behavior.
\end{DoxyWarning}

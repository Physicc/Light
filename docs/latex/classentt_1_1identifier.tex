\subsubsection*{template$<$typename... Types$>$\newline
class entt\+::identifier$<$ Types $>$}

Types identifiers. 

Variable template used to generate identifiers at compile-\/time for the given types. Use the {\ttfamily get} member function to know what\textquotesingle{}s the identifier associated to the specific type.

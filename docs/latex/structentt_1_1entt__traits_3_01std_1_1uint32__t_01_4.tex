\doxysubsection{Detailed Description}
Entity traits for a 32 bits entity identifier. 

A 32 bits entity identifier guarantees\+:


\begin{DoxyItemize}
\item 20 bits for the entity number (suitable for almost all the games).
\item 12 bit for the version (resets in \mbox{[}0-\/4095\mbox{]}).

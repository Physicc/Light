\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Entity$>$\newline
class entt\+::basic\+\_\+organizer$<$ Entity $>$}

Utility class for creating a static task graph. 

This class offers minimal support (but sufficient in many cases) for creating an execution graph from functions and their requirements on resources.~\newline
 Note that the resulting tasks aren\textquotesingle{}t executed in any case. This isn\textquotesingle{}t the goal of the tool. Instead, they are returned to the user in the form of a graph that allows for safe execution.

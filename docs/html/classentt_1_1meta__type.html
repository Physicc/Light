◆ reset()

void entt::meta_type::reset ( )
inline

Resets a type and all its parts.

This function resets a type and all its data members, member functions and properties, as well as its constructors, destructors and conversion functions if any.
Base classes aren't reset but the link between the two types is removed.

The type is also removed from the list of searchable types.
